Displacement

The simplest method, displacement coding moves each letter down the alphabet by the same numeric value. The obvious weakness with this method is brute force decoding, as only 26 unique combinations are available.

Letter Scramble

By switching each letter in the alphabet with a randomized letter of the alphabet, we increase our number of combinations to about 400000000000000000000000000 (26 factorial). While this is much more secure than the letter displacement from above, it is vulnerable to techniques such as searching for common words.
